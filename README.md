# PDFKIT

# PREVIEV `https://markuz899.github.io/pdfkit/`

Install the project dependencies with `npm`:

```bash
npm install
```

## Running

We are ready to launch the app! 🎉

To run the app in development mode:

```bash
npm start
```

To create a production build of the app and serve it:

```bash
npm run build
serve -s build
```

Enjoy 🍕
